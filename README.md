# MRI Software (mri-software)

MRI Software is a real estate technology platform serving over 45,000 clients across 170+ countries. It provides REST APIs and an open integration ecosystem for property management, investment management, lease accounting, facilities management, visitor and contractor management, and commercial real estate analytics. The MRI Information Exchange (MIX) hosts over 1,000 vetted APIs for data exchange between MRI's core platform and third-party or custom applications.

## APIs

### MRI Information Exchange (MIX) API

An open and connected ecosystem of secure REST APIs built by MRI and its partners that enables controlled data exchange between MRI's core property management and accounting platform and third-party or custom applications. Over 1,000 unique APIs operate within the MIX environment, all vetted by MRI engineers for security and integrity.

### MRI OnLocation REST API

A REST API for integrating with MRI OnLocation's people presence management platform. Enables programmatic access to visitor, contractor, and employee sign-in data, assets, certifications, locations, zones, inductions, and notifications. Supports OAuth 2.0 client credentials and API key authentication.

- **Human URL:** [https://apidocs.whosonlocation.com/](https://apidocs.whosonlocation.com/)
- **Base URL:** `https://login.whosonlocation.com`

### MRI Property Management X API

REST APIs providing programmatic access to MRI's core property management platform, covering residential and commercial property portfolios, units, leases, resident ledgers, maintenance work orders, and AP workflows. Authentication uses OAuth 2.0 with TLS 1.2+ encryption. Access is provisioned through the myMRI client portal.

- **Human URL:** [https://www.mrisoftware.com/mrifunctionalspecs/](https://www.mrisoftware.com/mrifunctionalspecs/)
